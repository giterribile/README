# README

# Anúncios de Vendas de Carros 

Aplicativo web desenvolvido com Streamlit para visualizar dados de anúncios de vendas de carros. O projeto permite explorar a distribuição da quilometragem dos veículos e a relação entre quilometragem e preço.

## Funcionalidades 

- Histograma da distribuição do odômetro dos veículos
- Gráfico de dispersão entre odômetro e preço
- Visualizações interativas com Plotly

## Acesse o aplicativo 

https://readme-2-1obn.onrender.com/

## Tecnologias utilizadas

- Python
- Pandas
- Plotly
- Streamlit

## Execução do projeto

O aplicativo realiza a leitura do arquivo de dados `vehicles_us.csv` e gera visualizações interativas:

- Histograma criado com `px.histogram`
- Gráfico de dispersão criado com `px.scatter`

## Resultados:

O histograma permite observar a distribuição da quilometragem dos veículos. A maior parte dos carros está concentrada na faixa de 0 a 200 mil quilômetros rodados, enquanto uma quantidade menor apresenta quilometragem mais alta.

O gráfico de dispersão permite analisar a relação entre quilometragem e preço, ajudando a observar como o valor dos veículos pode variar de acordo com o uso.
